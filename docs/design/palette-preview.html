Алерты — уровень кодируется ПАТТЕРНОМ, не только оттенком: error = filled («стоп») · warn = золотой ghost · live = crimson ghost + пульс · info = перивинкл ghost. Чинит «золото на золоте» в тёмной и разводит error↔live с первого взгляда. Все пары — WCAG AA, новых токенов 0.

Семантика (§9.2.5): error → hard-ошибка · secondary (золото) → soft-предупреждение · tertiary (crimson) → live/registration.

Главный CTA = secondary-container #EEC200 + тёмные чернила (не токен primary) — маппинг из заметок реализации.

Источник истины токенов — docs/design/design-system-dark.md и docs/design/design-system-light.md. Elevation в тёмной несёт тон+glow, в светлой — мягкую тень (см. панели «Компоненты → глубина»).
